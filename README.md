# ATE RAG KB Marketplace

This repository is the Claude Code plugin marketplace for
`WalterLuo/ate-rag-kb`.

## Install

In Claude Code:

```text
/plugin marketplace add WalterLuo/ate-rag-kb-marketplace
/plugin install ate-rag-kb@ate-rag-kb-marketplace
```

The plugin installs the ATE KB Router skill, which routes ATE engineering
questions to the local `ate_kb` MCP knowledge base first.

The full knowledge-base project, MCP server, installer, and documentation live
at:

https://github.com/WalterLuo/ate-rag-kb
